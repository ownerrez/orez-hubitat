# OwnerRez Hubitat Integration App
Integration module for connecting your Hubitat hub to your OwnerRez account.

## Installation
1. Download ZIP bundle from https://github.com/ownerrez/orez-hubitat/releases
2. From the Hubitat hub Main Menu, go to 'Bundles'
3. Click in the upper right hand corner, 'Import ZIP'
4. Choose the zip file downloaded in step 1
5. Click Import
6. Go to 'Apps'
7. Click 'Add User App'
8. Choose 'OwnerRez Integration'
9. On the OwnerRez Integration settings page, choose the locks you wish to expose to OwnerRez
10. Click Done (required step to finish the install)
11. Go back into the OnwerRez Integration app from the Apps menu
12. Click 'Connect to OwnerRez' button
13. Log into your OwnerRez account if not already, otherwise the window will popup and quickly disappear
14. You should now see a "You are connected to OwnerRez." message
15. Go to https://secure.ownerrez.com/settings/locks to finish the setup on OwnerRez
